描述圆周运动的物理量

物理量 定义、意义 公式
线速度v 描述圆周运动的物体运动快慢的物理量
矢量，方向和半径垂直，和圆周相切		 v=\frac{\Delta s}{\Delta t}=\frac{2\pi r}{T}
角速度 \omega 描述物体绕圆心转动快慢的物理量
矢量，但不研究其方向		 \omega=\frac{\Delta\theta}{\Delta t}=\frac{2\pi}{T}
周期 T 物体沿圆周运动一圈的时间
周期T的倒数为频率f(转速n)		 T=\frac{2\pi r}{v}=\frac{2\pi}{\omega}
f=n=\frac{1}{T}
向心加速度 a_{n} 描述线速度方向变化快慢的物理量
方向指向圆心		 a_{n}=\frac{v^{2}}{r}=\omega^{2}r

练习

  1. 匀速圆周运动是匀变速曲线运动 ( 错 )
  2. 物体做匀速圆周运动时，其线速度是不变的 ( 错 )
  3. 物体做匀速圆周运动时，其合外力是变力 ( 对 )
  4. 匀速圆周运动的向心加速度与半径成反比 ( 错 )

向心力

  1. 作用效果：向心力产生向心加速度，只改变速度的方向，不改变速度的大小
  2. 大小：F=m\frac{v^{2}}{r}=m\omega^{2}r=m\left( \frac{2\pi}{T} \right)^{2}r
  3. 方向：始终沿半径指向圆心，时刻在改变，即向心力是一个变力
  4. 来源：向心力可以由一个力提供，也可以由几个力的合力提供，还可以由一个力的分力提供。
注意事项
  1. 向心力是效果力，“物体受到向心力的作用”这种表述是不合适的，应改成“物体的向心力由某某力提供”；
  2. 中学阶段没有离心力这种表述。

生活中的圆周运动

练习

一同学表演荡秋千. 已知秋千的两根绳长均为 9 \;m，该同学和秋千踏板的总质量约为 50 \;kg. 绳的质量忽略不计. 当该同学荡到秋千支架的正下方时，速度大小为 6 \;m/ s，此时每根绳子平均承受的拉力约为 350\; N

答案

记每根绳上的拉力为 F_{T}，则

\begin{aligned} &a_{n}=\frac{v^{2}}{r}=4\;m /s^{2} \\ &2F_{T}-mg=ma_{n} \Rightarrow F_{T}=350\;N \end{aligned}
